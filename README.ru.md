# ATH CHECKER - Простой скрипт на Rust, для проверки падения токена от all-time-high цены (ATH)

## Для чего это нужно
Фильтр токенов по падению от ATH позволяет вывести статистику, стоит ли покупать определенный токен в данный момент. Особенно полезно для инвесторов в криптовалюту

## Использование:
Запустите файл **ath-checker.exe** (https://github.com/MrAlgoriZ/ath-checker/releases/latest), либо скомпилируйте:
```bash
cargo build --release 
```

После запуска инициализируется файл config.yaml в папке, в которой находится запущенный .exe файл

## Настройка
Вы можете поставить **"priority_tokens"**, заполняются они через знак "-". Это поле настраивает токены, которые проверяются в первую очередь

**"only_priority_tokens"** - поле которое позволяет запускать проверку только для выставленных **"priority_tokens"**.

**"filter_trash_tokens"** - фильтровать ли токены по волатильности. То есть проверять, не скам ли токен, и можно ли в него вкладываться (если значения мелкие, значит - токен мёртв)

**"token_check_interval_seconds"** - дополнительная задержка между проверками каждого токена (в секундах)

**"alert_on_priority_token"** - позволяет оповещать о падении **"priority_tokens"** в определенном интервале через телеграм. Настраиваются параметры: Включено ли (**enabled**); **timeout_minutes** (интервал в минутах); **chat_id** (ваш chat_id в телеграме); **telegram_token** (ваш токен бота, можно получить в https://t.me/BotFather)
